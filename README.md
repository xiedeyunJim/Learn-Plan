    http头部信息：
      Accpt:游览器能够处理的内容类型
      Accept-Encoding:游览器能够处理的压缩编码
      Accept-Charset:浏览器能够显示的字符集
      Accept-Langeuage:浏览器当前设置的语言
      Connection:浏览器与服务器之间连接的类型
      Cookie:当前页面设置的任何Cookie
      Host:发出请求的页面所在的域
      Referer:发出请求的页面的URL
      User-Agent:浏览器的用户代理字符串
